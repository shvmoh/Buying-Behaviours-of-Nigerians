# Buying-Behaviours-of-Nigerians-

*This dataset captures *consumer buying behavior* across different states and age groups in Nigeria.
It contains responses from shoppers describing:*

•⁠  ⁠*Demographics:* Gender, Age Range, Location.

•⁠  ⁠*Shopping Habits:* Frequency, Platforms used, Active vs. Occasional shoppers.

•⁠  ⁠*Financial Behavior:* Monthly spend, spend category, average spend.

•⁠  ⁠*Preferences:* Payment method, product types, buying decision factors.

### Total Respondents: (107)
